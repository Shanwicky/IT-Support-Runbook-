## 📌 Project Overview
This repository contains Compilation of 45 technical runbooks covering critical IT infrastructure, security, identity management, and systems troubleshooting. This portfolio was developed through deep research into corporate IT support structures and real-world employee desktop pain points as Student-Project.

The primary document can be referenced directly via IT_Support_Runbook_Portfolio.pdf).

## 🛠️ Home Lab Validation Environment
To bridge the gap between theory and practical engineering, the core identity, networking, and security runbooks in this portfolio were actively practiced and validated in a dedicated hypervisor environment:
* **Hypervisor:** VMware Workstation Pro
* **Directory Services:** Windows Server 2016 (Active Directory Domain Services, DNS, DHCP roles)
* **Client Environment:** Windows 10 Enterprise VMs utilizing dummy accounts to simulate corporate endpoints

## 📂 Portfolio Structure
Each runbook follows a strict, highly scannable corporate framework ensuring operational readiness:
1. **Symptoms:** Common end-user indicators and error codes.
2. **Root Causes:** Underlying technical triggers.
3. **Fix Steps:** Step-by-step remediation procedures.
4. **Verification:** Steps to confirm a successful fix.
5. **Rollback:** Safe procedures to revert changes if complications arise.
6. **Escalation:** Tiered routing guidelines to specialized teams (SOC, Network, Infrastructure).
